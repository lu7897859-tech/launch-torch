# Layer 0: Demand Validation

**Core file:** product.html

Capture purchase intent signals before building payment infrastructure.

## How it works
- Product page with clear two-form presentation
- Intent-grading form: 5 intent levels
- Submit via mailto: no server, no platform, no payment processor
- Each submission = a demand signal with timestamp + intent level

## Why it's unblockable
No platform involved. Pure HTML + mailto. No database, no API.
