# Layer 4: Static Dead Drops

**Core file:** docs/dead-drop.html

Platform-proof single-file distribution.

## How it works
Generate a single self-contained HTML document that contains your entire product pitch:
- All CSS inline (no external dependencies)
- All content in one file
- Fully responsive, works on any browser
- Can be hosted on ANY static file server

This file can be:
- Mirrored to any CDN
- Uploaded to any cloud storage
- Pinned to IPFS
- Attached to emails
- Copied to a USB drive and handed to someone

## Why it's unblockable
You can't block a file that exists on every server. The internet is built on copying files.
