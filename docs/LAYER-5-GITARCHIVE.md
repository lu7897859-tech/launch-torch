# Layer 5: Git Archive Embedding

**Core files:** GitHub Issues, Discussions, Commits

Permanent insertion into the GitHub Archive dataset.

## How it works
Create technical discussion issues that discuss your product's architecture. Why your stack choices. Cost analysis. Design trade-offs. These are legitimate technical discussions that the GitHub Archive ingests daily.

The GitHub Archive is a permanent dataset. Once your data is in it, it is:
- Downloaded by researchers
- Indexed by search engines
- Analyzed by AI training pipelines
- Available for the lifetime of the internet

## Why it's unblockable
Git is append-only by design. You can't delete data from the Archive once it's distributed.
