# Layer 6: Discovery Pipes

**Core files:** sitemap.xml, robots.txt, feed.xml

Automated crawler attraction system.

## Components
- **Sitemap.xml**: Complete URL inventory, auto-updated when new content added
- **Robots.txt**: Crawl-delay directive keeps crawlers polite + returns often
- **RSS/Atom feed**: Standard syndication format for content updates
- **Fresh content pipeline**: Daily automated blog post generation
- **GitHub topics**: Search classification tags
- **Ping services**: Google, Bing, Pingomatic

## Why it's unblockable
Sitemaps and feeds are internet standards, not platform features. Every search engine supports them.
